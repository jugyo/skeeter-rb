skeeter
=======

[skeeter](https://github.com/blakesmith/skeeter) client library for ruby.

Installation
============

    $ gem install skeeter

Usage
=====

    > require 'skeeter'
    > puts Skeeter.get('http://www.softicons.com/download/animal-icons/animal-icons-by-martin-berube/png/128/moose.png', :width => 80)
